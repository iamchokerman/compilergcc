## Dependencies

```
nasm
gcc-multilib
```

### How to install dependencies

* Ubuntu/Debian
```
sudo apt-get install nasm 
sudo apt-get install gcc-multilib
```

* Arch

```
sudo pacman -S nasm
sudo pacman -S gcc-multilib
```

## Usage


```./compile.sh```

to compile .asm files 

 ```./remove.sh```

to remove .o files after compiling
